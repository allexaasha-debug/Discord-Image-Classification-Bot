# Discord-Image-Classification-Bot
Bot Discord ini digunakan untuk mengklasifikasikan gambar ikan.
Bot akan menerima gambar dari pengguna, lalu menentukan apakah gambar tersebut Ikan koi, Ikan nemo, Atau bukan keduanya

bot ini bisa digunakan dengan cara:
1. Pengguna mengirim gambar dan ketikan $check melalui Discord.
2. Bot mengunduh gambar tersebut.
3. Gambar diproses.
4. Model Keras memprediksi kelas gambar.
5. Bot mengirim hasil ke chat.
